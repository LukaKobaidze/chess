## Getting Started

1. Clone the repo

```sh
git clone https://github.com/LukaKobaidze/chess
```

2. Install NPM/Yarn packages

```sh
npm install
```

or

```sh
yarn install
```

3. Start app

```sh
npm start
```
or
```sh
yarn start
```
